RTL-first UI for Arabic apps

A skill for AI coding agents that makes Arabic/RTL layout support automatic. Layout mirroring, CSS logical properties, Arabic typography, directional icons, animations, forms — handled.

Install the skill View on GitHub

Install

One command per agent. The skill drops into your agent's skills directory.

# Auto-detect Claude Code
npx skills add samuadda/rtl-skill -a claude-code

# Global (all projects)
npx skills add samuadda/rtl-skill -g
# Workspace-scoped
git clone --depth 1 https://github.com/samuadda/rtl-skill.git /tmp/rtl-skill
mkdir -p .agents/skills
cp -R /tmp/rtl-skill/skills/rtl .agents/skills/

# Global
mkdir -p ~/.gemini/antigravity/skills
cp -R /tmp/rtl-skill/skills/rtl ~/.gemini/antigravity/skills/
# Cursor / Windsurf / Cline — adapts to each agent's rules format
npx skills add samuadda/rtl-skill -a cursor
npx skills add samuadda/rtl-skill -a windsurf
npx skills add samuadda/rtl-skill -a cline
# Clone and copy to your agent's skill directory
git clone --depth 1 https://github.com/samuadda/rtl-skill.git
cp -R rtl-skill/skills/rtl .claude/skills/

Commands

All commands also work as natural language — "audit this for RTL issues", "rtlize this component".

/rtl-init

Scaffold a new project with RTL baked in — fonts, reset, config, base components.

/rtl-audit

Scan an existing codebase for RTL issues. Produces rtl-audit-report.md.

/rtl-convert <file>

Convert a specific component to RTL-aware.

/rtl-check

Quick golden-checklist pass on the current component.

What it covers

Arabic-language UI specifically. Other RTL scripts have different typography rules and aren't supported.

  • ✓ CSS logical properties (inline-start/end)
  • ✓ Arabic typography (Cairo / Tajawal, line-height, kerning)
  • ✓ Icon classification — knows what to flip
  • ✓ Component patterns (breadcrumbs, drawers, toasts, carousels)
  • ✓ Animations (slide direction, progress, skeletons)
  • ✓ Forms (alignment, labels, validation, number inputs)
  • ✓ Tables (column order, sort indicators)
  • ✓ LTR islands (numbers, URLs, code in Arabic text)
  • ✓ Dynamic LTR/RTL switching
  • ✓ Tailwind, plain CSS, CSS-in-JS, React Native
